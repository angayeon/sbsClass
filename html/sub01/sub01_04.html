웹 접근성(Web Accessibility)

웹 접근성은 모든 사용자가 신체적, 환경적 조건에 관계없이 웹에 접근하여 이용할 수 있도록 보장하는 것을 말합니다.

즉, 어떠한 사용자(일반인, 장애인, 고령자 등)가 어떠한 환경(브라우저, 운영체제, 디바이스의 환경이나 사양 등)에서도 어려움 없이 접근, 이용할 수 있는 것을 접근성이라고 합니다.

접근성 > 마크업(문법), 반응형(디바이스의 환경 -사이즈), 호환성(브라우저 운영체계 등 호환 가능한지 여부) 등

WAI(Web Accessibility Initiative)

웹의 표준화 관련 국제 기구인 W3C(World Wide Web Consortium)에서는 1990년대 중반 이후 폭발적으로 성장한 웹 서비스에서 장애인의 접근성에 관련된 문제가 발생하자,

1997년에 웹 접근성 이니셔티브(WAI : Web Accessibility Initiative)라는 산하 단체를 설립하여 이 문제를 전문적으로 연구하기 시작했습니다.

WAI의 웹 콘텐츠 접근성 지침은 다음과 같습니다.

1. 인지성(Perceivable) - 정보와 사용자 인터페이스 요소는 그들이 인지할 수 있도록 사용자에게 표시될 수 있어야 한다.

2. 운용성(Operable) - 사용자 인터페이스 요소와 탐색은 운용 가능해야 한다.

3. 이해성(Understandable) - 정보와 사용자 인터페이스 운용은 이해할 수 있어야 한다.

4. 견고성(Robust) - 콘텐츠는 보조 기술을 포함한 넓고 다양한 사용자 에이전트에 의존하여 해석될 수 있도록 충분히 내구성을 가져야 한다.

오픈왁스(OpenWAX)

웹 접근성을 확인하기 위해 여러가지 툴을 사용할 수 있는데 가장 기본적으로 많이 쓰이는 툴이 오픈왁스입니다. 오픈왁스는 크롬과 파이어폭스에서 사용할 수 있는 도구입니다.

1. 크롬 우측 상단 제일 끝 버튼 클릭

2. 도구 더보기 > 확장 프로그램 클릭

3. 확장 프로그램이 열리면 ‘더 많은 확장 프로그램 다운로드’ 클릭

4. 크롬 웹 스토어가 열리면 좌측상단에 ‘스토어 검색’ 창에다가 ‘openwax’라고 검색
