텍스트 관련 태그

HTML 문서는 텍스트로 구성되어 있습니다. 웹 문서의 가장 기본적인 텍스트 태그의 종류를 살펴보고 해당 태그들이 어떤 속성을 갖고 있는지 배워봅니다.

자세히보기

목록 태그

1. <ul> 태그 순서가 없는 목록 태그

목록 태그의 초기값은 항목 앞에 동그라미 또는 사각형 같은 불릿이 표시됩니다.

(Section03)-p.065~

<!-- 순서가 없는 리스트 -->

<ul>

<li>짜장면</li>

<li>짬뽕</li>

<li>탕수육</li>

<li>야끼우동</li>

<ul>

2. <ol> 태그 순서가 있는 목록 태그

순서가 있는 목록 태그의 초기값은 숫자로 표시됩니다.

(Section03)-p.066~

<!-- 순서가 있는 리스트 -->

<ol>

<li>영어</li>

<li>음악</li>

<li>수학</li>

<li>과학</li>

<ol>

3. <dl>, <dt>, <dd> 태그 특정 용어에 대한 정의와 설명을 나타내는 태그

(Section03)-p.067~

<!-- 정 용어에 대한 정의와 설명 -->

<dl> <!-- dl(data list) -->

<dt>강남지점</dt> <!-- dt(data title) -->

<dd>주소 : 서울특별시 서초구 서초대로 77길 3</dd> <!-- dt(data dedail) -->

<dd>전화 : 02)000-2222</dd>

<dl>

주석문 표현

주석문은 실행 결과에 영향을 주지 않습니다. 따라서 코드를 쉽고 빠르게 수정·보완 하기 위해 작업시 표시하는 방법입니다.

(Section03)-p.064~

<!-- 주석문 (단축키: ctrl + /) -->

<!-- 문서의 내용 시작 -->

<h1>본문의 제목 1</h1>

<p>본문의 내용이 들어갑니다. </p>

<!-- 문서의 내용 끝 -->

주석문은 작업자가 원하는 곳에 ctrl+/ 을 사용하여 넣을 수 있습니다.
