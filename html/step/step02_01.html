이미지태그

HTML 문서에는 img 태그를 사용하여 이미지를 표현할 수 있습니다.

이미지 저장 방법

이미지를 저장할 때, 반드시 소문자로 저장, 공백 사용 불가, 한글파일명으로 사용 불가

웹 문서가 있는 폴더 내에 불러오려는 이미지 파일이 존재해야 합니다.

“img” 나 “images” 와 같은 영문 폴더를 만들어 별도로 관리하는 것이 좋습니다.

이미지 태그 속성

이미지 alt값은 웹 표준에 따라서 반드시 속성에 넣어야 합니다.

<!-- img태그 (단축키: img + tab키) -->

<img src="이미지 파일의 경로 또는 url" alt="이미지에 대한 설명" ></img>

속성, 설명 정보제공
속성 설명

src

 이미지의 경로 지정

alt

 이미지가 보이지 않을 때 표시하는 대체 글자 지정 (즉, 이미지에 대한 설명)

src="이미지의 경로를 적는다!" 제일 중요!

alt 값은 생략 금지! 무조건! 이미지에 대한 설명을 써줍니다.

이미지 alt값 예시

alt="아보카도 사진"

alt="강아지 사진"

alt="도너츠 사진"

이미지의 절대 경로와 상대 경로

이미지를 불러올 때 가장 중요한 것은 이미지의 경로입니다.

1) 절대 경로

어떠한 웹 페이지나 파일이 가지고 있는 고유한 경로를 절대 경로라고 합니다.

웹페이지에서 이미지의 URL을 가지고 오거나 내 컴퓨터 안의 이미지 경로를 가져올 때 절대경로를 사용하는 것 입니다.

웹페이지 이미지 주소를 불러오는 경우 : <img src="http://www.xxxxx.com/image.jpg">

내 컴퓨터 안의 이미지 경로를 불러오는 경우 : <img src="C:\user\...\image.jpg">

(Section04)-p.076~

2) 상대 경로

작업중인 공간(폴더)을 기준으로 상대적인 위치를 가리키는 경로를 상대 경로라고 합니다.

같은 폴더 안의 이미지 사용 : <img src="image.jpg">

하위 폴더 안의 이미지 사용 : <img src="file/image.jpg">

상위 폴더 안의 이미지 사용 : <img src="../file/image.jpg">

이미지의 파일이 같은 폴더에 저장되어있을 경우 파일명만 적어줍니다.

하위 폴더에 저장되어 있는 경우에는 "폴더명/이미지파일" 로 적어줍니다.

상위 폴더에 저장되어 있는 경우에는 "../폴더명/이미지파일" 로 적어줍니다.
