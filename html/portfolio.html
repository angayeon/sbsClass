웹포트폴리오 과정

웹포트폴리오 과정은 Photoshop, Illustrator, HTML, CSS, Java Script, J-Query를 배웠다는 전제 하에 수업을 진행합니다.

수업 목표

전체적인 웹사이트의 기획부터 구축까지 진행하며, 작업물의 UI 코딩가이드를 통해 완성도 높은 포트폴리오를 만들어봅니다.

수업 수준 및 난이도

1. 난이도 : 상 ★★★★★ (퍼블리싱을 모르는 수강생들은 수업 불가)

2. 선수강과목 : 포토샵, 일러스트, 웹1, 웹2, 웹3

3. 사용 프로그램 : Visual Studio Code 또는 Brackets(무료 코딩 에디터 사용)

4. 기타 보조 프로그램 : 포토샵, 일러스트

수업 방향

1. 개인별 HTML, CSS, JS 레벨 진단 및 그에 따른 포트폴리오 전략 수립

2. 원하는 포트폴리오 스타일 서치

3. 밴치 사이트 분석 및 자료 조사

4. 그리드 가이드에 맞춰 시안 제작

5. 실무에서 주로 쓰이는 이슈 파악

6. UI 코딩가이드를 통한 포트폴리오 구축

7. 포트폴리오 발표 및 공유

취업 분야
