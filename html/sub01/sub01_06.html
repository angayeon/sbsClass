캐스캐이딩 스타일시트(CSS)

CSS에서 C는 Cascanding의 약자로 위에서 아래로 흐르는 스타일시트라는 뜻입니다.

캐스캐이딩의 의미를 이해하고 스타일의 우선 순위에 대한 개념을 이해한다면 효율적인 웹 문서를 만들 수 있습니다.

1) 스타일 우선 순위

우선 순위는 어떤 스타일을 먼저 적용할 것인지 결정하는 규칙을 말합니다.

중요도 : 중요 스타일은 !important를 붙여서 나타냅니다.

브라우저의 기본 스타일시트 : 브라우저에서 따로 스타일을 지정하지 않아도 나타나는 스타일을 말합니다.

/* 주석문 (단축키: ctrl + /) */

p { color : red !important ; text-align : center ; } /* 글자 스타일 */

p { color : blue ; } /* 글자 색상 */

설명

!important 는 우선순위가 높다.

인라인 스타일 > id 스타일 > class 스타일 > 태그 스타일

2) 소스 우선 순위

같은 태그나 class등에 스타일을 준 경우, 마지막에 적용된 스타일을 우선순위로 봅니다.

/* 주석문 (단축키: ctrl + /) */

p { color : red; font-weight : bold ; } /* 글자 스타일 */

p { color : blue ; } /* 글자 색상 */

마지막에 적용된 스타일의 우선순위가 높다.
