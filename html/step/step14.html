변형 속성(Transform)

CSS3의 transform속성을 이용하면 다양한 2차원, 3차원 변환이 가능합니다.

1. 2차원 변환

프로그램에서 화면의 좌표는 X축과 Y축을 사용합니다. 이 때 X좌표, Y좌표가 있는 화면을 2차원 화면 좌표라고 합니다.

1) 2차원 변환 함수

변환 함수, 설명, 방법, 예시 정보 제공
변환 함수 설명 방법
tanslate(tanslateX, tanslateY) 특정 크기만큼 이동 %나 px로 표시
transform: translate(-50%, -50%);
tanslateX X축으로 특정 크기만큼 이동합니다. %나 px로 표시
transform: translateX(30px); x축으로 30px 이동
tanslateY Y축으로 특정 크기만큼 이동합니다. %나 px로 표시
transform: translateY(-100px); y축으로 -100px 이동
scale(scaleX, scaleY) 특정 크기 만큼 확대 및 축소합니다. 크기는 1을 기준으로 100%
1보다 작으면 0.1~0.9 축소
1보다 크면 1.1~ 확대
transform: scale(1.5); x축 y축 전체 1.5배 확대
transform: scale(2, -1.2); x축 2배 확대, y축 1.2배 축소
transform: scale(-0.5, 5); x축 0.5배 축소, y축 5배 확대
rotate 특정 각도만큼 회전합니다. 시계방향(deg) 또는 시계반대방향(-deg) 사용
transform: rotate(180deg); 시계방향으로 180도 회전
transform: rotate(-135deg); 시계반대방향으로 135도 회전
skew 특정 각도만큼 기울입니다. 시계방향(deg) 또는 시계반대방향(-deg) 사용
transform: skew(45deg, 15deg); 시계방향으로 x축 45도, Y축15도 기울이기
transform: skew(-135deg) 시계반대방향으로 135도 기울이기
