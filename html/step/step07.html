다양한 CSS 선택자

선택자를 사용해 특정 태그를 선택하면 해당 태그에 원하는 스타일과 기능을 적용할 수 있습니다.

1. 전체 선택자

선택자, 형태, 설명, 활용 예 정보제공
선택자 형태 설명 활용 예
전체 선택자 *
  • 모든 웹 페이지에 빠지지 않고 사용하는 선택자
  • 웹 페이지 시작 전 초기화(노말라이즈) 시키기 위해 사용
  • 부모앨리먼트 내에 자식 앨리먼트 전체를 선택할 때도 사용

* { 속성: 속성값; }
div > * { 속성: 속성값; }
.list-box > * { 속성: 속성값; }

2. 태그 선택자

선택자, 형태, 설명, 활용 예 정보제공
선택자 형태 설명 활용 예
태그 선택자 div, h1, p, span... 특정 태그를 선택합니다. div { 속성 : 속성 값; }

3. 아이디 선택자와 클래스 선택자

선택자, 형태, 설명, 활용 예 정보제공
선택자 형태 설명 활용 예
아이디 선택자 #아이디명 아이디 속성을 가지고 있는 태그를 선택합니다. #ui-accordion-01 { 속성 : 속성 값; }
클래스 선택자 .클래스명 특정 클래스를 가지고 있는 태그를 선택합니다. .txt-box { 속성 : 속성 값; }

4. 자손 선택자와 후손 선택자

선택자, 형태, 설명, 활용 예 정보제공
선택자 형태 설명 활용 예
자손 선택자 선택자 A > 선택자 B 선택자 A의 자손에 위치하는 자손 선택자 B 태그를 선택합니다. div > p { 속성 : 속성 값; }
후손 선택자 선택자 A 선택자 B 선택자 A의 후손에 위치하는 후손 선택자 B 태그를 선택합니다. section div { 속성 : 속성 값; }

5. 동위 선택자

동위 선택자는 동위 관계에서 뒤에 위치한 태그를 선택할 때 사용하는 선택자입니다. (형제 관계 중 뒤에 나오는 동생 혹은 동생들 전체 선택)

선택자, 형태, 설명, 활용 예 정보제공
선택자 형태 설명 활용 예
동위 선택자 선택자 A + 선택자 B 선택자 A의 바로 뒤에 위치하는 선택자 B 태그를 선택합니다. div + a { 속성 : 속성 값; }
선택자 A ~ 선택자 B 선택자 A의 뒤에 위치하는 선택자 B 태그(모두)를 선택합니다. section ~ div { 속성 : 속성 값; }

6. 구조 선택자

선택자, 형태, 설명, 활용 예 정보제공
선택자 형태 설명 활용 예
구조 선택자 선택자:first-child 부모 박스 안의 형제 관계 중 첫 번째에 위치하는 태그를 선택합니다. div > p:first-child { 속성 : 속성 값; }
선택자:last-child 부모 박스 안의 형제 관계 중 마지막에 위치하는 태그를 선택합니다. div > img:last-child { 속성 : 속성 값; }
선택자:nth-child(n)
  • 부모 박스 안의 형제 관계 중 앞에서 수열 n번째에 해당하는 태그를 선택합니다.
  • 부모 태그와 자식 태그가 동일하면 부모 태그 또한 계산됩니다.
 div > div:nth-child(3) { 속성 : 속성 값; }
선택자:nth-last-child(n)
  • 부모 박스 안의 형제 관계 중 뒤에서 수열 n번째에 해당하는 태그를 선택합니다.
 div > span:nth-last-child(3) { 속성 : 속성 값; }
수열이란?

자연수를 정의역으로 하는 함수를 말합니다. 여기서 (n)은 0부터 숫자가 대입된다고 생각하면 됩니다. 예를 들면 2*0 / 2*1 / 2*2... 이렇게 계산이 이루어집니다.

자주 사용하는 수열 중에서는 숫자 번째(숫자), 짝수열(2n), 홀수열(2n+1)이 있습니다.

수열 : 2n+1(숫자도 입력 가능)

7. 형태 구조 선택자

선택자, 형태, 설명, 활용 예 정보제공
선택자 형태 설명 활용 예
형태 구조 선택자 선택자:first-of-type 부모 박스 안의 형제 관계 중 첫 번째로 등장하는 특정 태그를 선택합니다. div > span:first-of-type { 속성 : 속성 값; }
선택자:last-of-type 부모 박스 안의 형제 관계 중 마지막으로 등장하는 특정 태그를 선택합니다. div > p:last-of-type { 속성 : 속성 값; }
선택자:nth-of-type(n)
  • 부모 박스 안의 형제 관계 중 앞에서 수열 n번째에 해당하는 특정 태그를 선택합니다.
  • 부모 태그와 자식 태그가 동일하면 부모 태그 또한 계산됩니다.
 div > div:nth-of-type(3) { 속성 : 속성 값; }
선택자:nth-last-child(n)
  • 부모 박스 안의 형제 관계 중 뒤에서 수열 n번째에 해당하는 특정 태그를 선택합니다.
 div > span:nth-of-type(3) { 속성 : 속성 값; }
구조 선택자 vs 형태 구조 선택자

형태 구조 선택자는 구조 선택자와 비슷하지만 특정 태그의 형태를 구분합니다.
