HTML5의 시작

HTML이란?

사용자들은 웹 사이트(홈페이지)에 접속하여 정보를 얻는데 이때 눈으로 보면서 얻는 정보의 형태는 대부분 HTML 언어로 작성되어 있습니다.

HyperText Markup Language의 약자로 인터넷에서 클릭을 통해 자유롭게 이동하면서 해당 정보를 확인할 때 보는 웹 문서입니다.

(Section01)-p.017~

HTML문서의 기본 구조와 문법 이해

HTML문서에서 활용되는 태그는 <>로 묶어서 표현합니다.

태그란? 웹 페이지에 나타낼 내용을 전달하는 언어입니다. 태그 입력 시 정확하게 사용해야하고 일관성을 유지하기 위해 영문 소문자로 작성하는 것이 좋습니다.

태그 - 태그 작성 방법, 태그 사용 예시 정보제공
태그 작성 방법 태그 사용 예시

<태그 명령어>

<html>, <head>, <title>, <body>, <footer>, <table>, <h1>, <p>, <img> ... etc

태그는 <>로 묶어서 표현

태그를 작성할 때는 문법대로 작성하고 소문자로 입력

잘못된 표현 : <HTML>, {html}, [Html]

HTML문서 만들기

태그를 활용하여 HTML문서를 만들어봅니다.

저장할 때에도 영문 대문자와 공백은 넣지 않으며, 특수 문자는 ‘_(언더바), -(하이픈)’ 이외에는 사용하지 않도록 주의합니다.
