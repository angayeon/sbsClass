하이퍼링크

모든 웹 페이지는 서로 연결되어 있습니다. 방문자가 원하는 정보를 확인하려고 클릭하는 순간 링크를 통해 다양한 웹 페이지로 이동합니다. 이것을 바로 하이퍼텍스트 링크라고 부릅니다.

링크속성

링크는 a로 시작합니다. 보통 a 링크라고 하며 앵커 태그라고도 합니다.

<!-- a태그 (단축키: a + tab키) -->

<a href="이동할 페이지 또는 url" ></a>

<a href="#" target="_blank" title="새창열림" ></a>

속성, 설명 정보제공
속성 설명
target="_self"

현재 페이지 창에 링크 페이지가 나타납니다. (바로 이동)
target="_blank"

현재 페이지는 그대로 두고, 새로운 창(탭)에 링크가 열립니다.
target="_parent"

프레임으로 나누어진 페이지 전체를 부모 프레임으로 페이지가 이동합니다.
(거의 사용 안함 / iframe을 사용할 경우 간혹 사용)
title="설명"

target 속성을 _blank로 지정했다면 title 속성은 꼭 넣어줘야 합니다. title 속성은 링크 위로 마우스를 가져가면 말풍선 형태로 문구가 나타납니다.

예) <a href="링크주소" target="_blank" title="네이버로 새창 열림"></a>

href="url주소를 적는다" 제일 중요!

타겟 블랭크를 사용했으면 타이틀은 세트로 같이 다닌다! 무조건 새창열림에 대한 문구를 넣어줘야한다.
