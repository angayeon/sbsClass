내부 스타일시트

내부 스타일시트는 웹 문서 안에 사용할 스타일을 정리한 것입니다.

쉽게 말해, <style> </style> 태그 안에 스타일을 정리하고 html 문서의 <head> </head> 태그 안에 <style> </style> 태그를 넣는 것을 말합니다.

<!DOCTYPE html>

<html lang="ko">

<head>

<style>

/* 내부 스타일시트 시작 */

div { text-align : center ; }

/* 내부 스타일시트 끝 */

</style>

</head>

<body>


</body>

</html>

설명

잊지말기!

내부 스타일시트는 꼭 <head> </head> 태그 안에 넣는다!
