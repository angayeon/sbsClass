다양한 스타일 속성 (기본)

스타일에 관련된 속성들은 매우 다양합니다. 기본적으로 사용되는 스타일의 종류에 대해 알아봅니다.

1) 글자 속성

font-size 16px / 3rem / 9pt ...

font-weight 100~300(가늘게) / 400~600(중간) / 700~900(굵게) / lighter / bold / normal ...

color red, blue(색상명) / #999999, #efefef, #0a6eb4 (#색상 코드) / rgb(색상값) ...

이 밖에 font-family, font-sytle, line-hight 등이 있습니다.

2) 문단 속성

display none / block / inline-block / inline 등

text-align center / left / right

word-spacing 숫자 입력 (px, % 가능)

letter-spacing 숫자 입력 (px, % 가능)

3) 백그라운드 속성

background-color 색상명, #색상코드, rgb(색상값) ...

background-image url(이미지 경로)

background-size 100% / cover, contain, px값

background-repeat no-repeat / repeat-x / repeat-y

이 밖에 background-attachment, background-position 등이 있습니다.

4) 박스 속성

width 숫자 입력 (px, % 가능)

height 숫자 입력 (px, % 가능)

padding 숫자 입력 (px, % 가능)

margin 숫자 입력 (px, % 가능)

border 테두리 굵기, 테두리 스타일, 테두리 색상 지정 ex) 2px solid black / 3px dashed #000
