block 태그 글자를 한줄로 표현하는 block 형태의 태그

1. 제목 태그

제목을 나타내고 구분하는 태그

<h1> Header-1 </h1>

<h2> Header-2 </h2>

<h3> Header-3 </h3>

<h4> Header-4 </h4>

<h5> Header-5 </h5>
<h6> Header-6 </h6>

자동 볼드체

숫자가 커질수록 글자의 크기가 작아진다.

크기를 위해 사용하지 말 것! => 글자 크기는 CSS를 이용해 지정해준다.

2. 글(단락) 태그

본문의 단락으로 구분하는 태그

<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Itaque, vitae, animi! Corporis, perspiciatis totam. Laborum mollitia odit, amet sapiente corporis quam ab error at quae voluptatum, ipsa pariatur nobis est. </p>

3. 이외 태그

<br> : 줄바꿈

<hr> : 수평줄

<pre> ~ </pre> : 공백이나 줄 바꿈 그대로 적용 -> 나중에 동영상 자막넣을 때 유용하게 쓰임

Inline 태그 한 줄로 줄 바꿈 없이 텍스트를 표시하는 inline 형태의 태그

4. <strong>, <b> 태그

글자를 굵게 표시할 때 사용하는 태그

<strong> 굵게 </strong>
<b> 굵게 </b>

5. <i>, <em> 태그

<i> 기울이기 </i>
<em> 기울여서 강조 </em>

i = 이탤릭체(Italic)

em = 강조(emphasize)

6. <span> 태그

별 다른 표시가 나타나진 않지만, 특정 텍스트만 특정 스타일을 적용하고자 할 경우 <span> 태그로 묶어서 사용

<p><span> 특정 스타일을 적용 </span>할 때 사용</p>

7. 이외 태그

<q> ~ </q> : 인용문

<mark> ~ </mark> : 형광펜 효과 내기
