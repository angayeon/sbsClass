외부 스타일시트

내부 스타일의 단점 : html 문서를 불어와서 제일 상단에 넣어야하기 때문에, 문서가 길어지거나 페이지 수가 많아지면 관리하기가 힘들고, 코드 수정시에도 효율성이 떨어집니다.

따라서 HTML 문서와 스타일 문서(CSS) HTML 문서와 스타일 문서(CSS), 스크립트 문서(JS)를 별도의 폴더나 파일로 저장해서 관리하는게 효율적입니다.

<!DOCTYPE html>

<html lang="ko">

<head>

<link rel="stylesheet" href="외부 스타일 파일 경로" type="text/css">

/* ex) /css/style.css 또는 ../../sub_style.css 등 으로 경로 표현 */

</head>

<body>


</body>

</html>

잊지말기!

link + tab 버튼 누르면 <link rel="stylesheet" href=""> 태그 자동 생성!

외부 스타일시트는 꼭 <head> </head> 안에 <link rel="stylesheet" href=""> 태그로 연결시킨다!
