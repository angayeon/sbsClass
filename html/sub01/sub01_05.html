CSS3의 시작

CSS이란?

CSS는 브라우저 화면에 다양한 디자인과 구성을 나타내기 위해 사용되는 가장 기본적인 도구 입니다.

Cascanding Style Sheets의 약자로 HTML과 함께 웹 표준의 기본 개념입니다.

HTML 태그가 웹 문서의 뼈대(구조)를 구성하고 있다면 CSS(스타일시트)는 텍스트의 색상을 비롯하여 태그의 크기, 위치, 레이아웃 등 다양한 디자인 요소를 담당하고 있습니다.

스타일 + 스타일 + 스타일 + ··· + ··· => 스타일 시트 / 스타일들을 모아 놓은 것을 스타일 시트라고 합니다.

CSS 기본 선택자의 이해와 형식

1) 선택자의 종류

태그 선택자 : html 태그에 스타일을 적용하는 태그 선택자 / 태그에 직접적으로 지정하는 방식

아이디 선택자 : id 속성으로 묶은 부분만 스타일을 적용하는 id 선택자 / #아이디명, id="아이디명"

클래스 선택자 : class 속성으로 묶은 특정 부분만 스타일을 적용하는 선택자 / .클래스명, class="클래스명"

설명

2) 속성과 값 그리고 문법

CSS는 문법적으로 { } 로 묶어서 표현합니다.

태그 - 태그 작성 방법, 태그 사용 예시 정보제공
스타일 작성 방법 스타일 사용 예시

선택자 { 속성 : 값 ; }

h1 { color : orange; }

선택자(태그, id, class) 뒤에 { }

{ }안에 스타일을 표현

속성은 콜론( : )으로 구분, 값은 세미콜론( ; )으로 구분

3) 스타일 주석

스타일시트의 주석문 역시 html의 주석문과 같이 작업자의 이해를 돕기 위해 주석문을 사용할 수 있습니다.

(section05)-p.086~

/* 주석문 (단축키: ctrl + /) */

h1 { color : red ; } /* 제목 태그의 글자 색상 */

p { text-align : center ; } /* 문단의 정렬 방식 */

html의 주석은 <!-- 이렇게 생겼고 -->

css의 주석은 /* 이렇게 생겼다 */

/* ~~ */ 이 사이에 내용을 입력한다

주석모양은 안외워도 돼요! ctrl+/ 만 기억합시다!

CSS 스타일시트

1) 내부 스타일시트

내부 스타일시트는 웹 문서 안에 사용할 스타일을 정리한 것입니다.

쉽게 말해, <style> </style> 태그 안에 스타일을 정리하고 html 문서의 <head> </head> 태그 안에 <style> </style> 태그를 넣는 것을 말합니다.

2) 외부 스타일시트

내부 스타일의 단점 : html 문서를 불어와서 제일 상단에 넣어야하기 때문에, 문서가 길어지거나 페이지 수가 많아지면 관리하기가 힘들고, 코드 수정시에도 효율성이 떨어집니다.

따라서 HTML 문서와 스타일 문서(CSS) HTML 문서와 스타일 문서(CSS), 스크립트 문서(JS)를 별도의 폴더나 파일로 저장해서 관리하는게 효율적입니다.

인라인 스타일

인라인 스타일은 html 문서 내부에서 간단한 스타일을 지정하고자 할 경우 스타일시트를 사용하지 않고 스타일 대상에 직접 적용하는 경우를 말합니다.

해당 태그에 style 속성을 사용해 <태그 style="속성 : 값;"> 형태로 사용합니다.

/* 태그 안에 스타일 속성을 직접 넣어주는 것 */

<div style="text-align: center;">

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Dolore quam, suscipit deserunt doloribus ipsa fuga ab, molestias quo amet temporibus dignissimos vitae, accusamus aspernatur tenetur id similique fugit beatae. Officia.

</div>

인라인 스타일은 최대한 안쓰는 방향으로 한다!
